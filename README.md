🎮 Contexte

Projet de site web dédié à une communauté RP sur Fallout 76, incarnant une faction inspirée de la Confrérie de l'Acier.

La communauté regroupe des joueurs PlayStation pratiquant le roleplay militaire dans l’univers post-apocalyptique des Appalaches.

🛡️ Présentation

La Confrérie de Virginie-Occidentale est une organisation RP structurée autour :

d’une hiérarchie militaire stricte
d’un système de régiments spécialisés
d’un lore évolutif et immersif
d’un codex servant de doctrine

Objectif :

Créer une expérience RP immersive, organisée et évolutive.

🎯 Objectifs du site
Présenter la communauté
Centraliser le lore
Expliquer l’organisation (rangs, régiments)
Faciliter le recrutement
Renforcer l’immersion (design Fallout / militaire)
